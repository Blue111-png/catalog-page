# 📖 Pr# 📖 Product Catalog Page

A full‑stack catalog application built with **React (frontend)**, **Express (backend)**, and **PostgreSQL (database)**.  
Features:
- Display products in a professional catalog layout
- Upload new products with images
- Manage product details dynamically

---

## 🚀 Tech Stack
- **Frontend**: React
- **Backend**: Express + Node.js
- **Database**: PostgreSQL (dockerized)
- **Image Hosting**: [Cloudinary](https://cloudinary.com/) — simple, free tier, easy API integration
